# Lap-Trinh-Mang
Khi chạy project trong Visual Studio, đoạn mã sẽ chỉ chạy một phần (Server hiển thị trước) và không khởi chạy client
Để phù hợp logic khởi tạo client tự nhiên, hãy mở file UDP Client.exe bên ngoài để thêm nhiều client cùng lúc
Nếu như không tìm thấy file .exe trên hãy theo đường dẫn sau /UdpChatApp - BTL/UdpChatApp/bin/Debug/net8.0-windows/UdpChatApp Client.lnk
